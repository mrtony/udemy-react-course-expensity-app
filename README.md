# Expenses App

- develop setup
- build setup
- git commands

## develop
```
npm i
npm run serve
```

## build code to /build
```
npm run build
```

## Git Commands

- git init: Create a new git repo
- git status:   View the changes in project code
- git add: Add files to staged area
- git cimmit: Create a new commit with files from stage area
- git log: View recent commit
